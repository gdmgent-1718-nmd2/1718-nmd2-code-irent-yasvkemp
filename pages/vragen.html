Veelgestelde vragen

Betalen
Hoe krijg ik een promotiecode?

Op speciale gelegenheden zoals je verjaardag krijg je een promotiecode toegestuurd via je e-mailadres als je bij ons bent ingelogd.
Dit is persoonlijk naar jou verstuurd en kan dus maar 1 keer gebruikt worden.

Hoe gebruik ik een promotiecode?

Je kan je promotiecode invoeren tijdens het betalen. Na het kiezen van een datum en uur klik je op ‘verder met bestellen’ daar kan je je gegevens invullen daarna klik je verder op ‘bevestigen en betalen’
Je krijgt dan een overzicht van je gegevens en daar kun je na ‘vul de promotiecode in:’ je promotiecode invoeren. Deze word automatische verrekend.

Wat als ik mijn promotiecode ben vergeten invullen?

Dit kan niet meer verrekend worden in een al betaalde reservatie maar de code blijft geldig. Je kan hem dus gebruiken bij een volgende reservatie.

Kan ik twee promotiecodes gebruiken bij mijn reservatie?

Neen, je kan maar één promotiecode gebruiker per reservatie. Maar je andere promotiecode blijft geldig. Je kan hem dus gebruiken bij een volgende reservatie.

Over Vr
Wat is Virtual Reality?

Virtual Reality is een spel/beleving waarbij je met een speciale bril je in een andere wereld waant. Je kunt om je heen kijken waarbij het speelveld met je meedraait. Je lijkt midden in een computerspel te zijn of een paar honderd meter boven de grond over een plank te lopen.

Hoe werkt Virtual Reality?

Je krijgt een Samsung gear VR-bril op, door deze bril waan je jezelf in de omgeving welke op dat moment geprojecteerd word. Door de koptelefoon krijg je ook alleen nog de geluiden uit de geprojecteerde omgeving mee. Doordat de belevenis zo echt aanvoelt, ervaar je alsof je echt in de geprojecteerde wereld rond loopt.

Voor wie is Virtual Reality geschikt?

Virtual Reality is geschikt voor iedere leeftijd tussen 8 en 100 jaar. Zowel voor digibeten als voor gamers!

Is het ook leuk als je niet van video games houdt?

Absoluut, virtual reality is een ontzettend gave ervaring welke je een keer geprobeerd moet hebben.

Mijn gegevens
Wachtwoord vergeten?

Je kan je wachtwoord herinstellen via ‘Mijn account’.
Klik op ‘wachtwoord wijzigen’ en vul je e-mailadres in.
We sturen je een e-mail met instructies om je wachtwoord opnieuw in te stellen.

Hoe kan ik mijn e-mailadres wijzigen?

Je kan je e-mailadres wijzigen door je te registreren als nieuwe klant met een nieuw e-mailadres en een nieuw wachtwoord. Je oude account blijft natuurlijk bestaan voor het bekijken van je eerdere bestellingen.
Wij kunnen je oude en nieuwe account niet samenvoegen.

Ik ben verhuisd, hoe kan ik mijn adres wijzigen?

Je kan je adres wijzigen via ‘Mijn account’. Klik op ‘adres wijzigen’ en klik vervolgens op ‘opslaan’.

Mijn reservatie
Kan ik mijn reservatie nog aanpassen?

Je kan deze tot een week voor de reservatie aanpassen door ons een e-mail te sturen via ‘Contact’ en daar het contactformulier in te vullen.
Is jouw reservatie minder dan een week verwijderd van vandaag dan kan u een nieuwe reservatie maken maar u zal de kosten moeten betalen van u geannuleerde afspraak.

Kan ik twee reservaties maken?

Dit kan. Maak uw eerste reservatie en volg de stappen tot de betaling. Na het betalen van uw eerste reservatie kan u klikken op ‘reserveren’ daarna kan u een datum en een uur kiezen voor uw volgende reservatie en de stappen volgen tot aan de betaling.
U zal per reservatie een bevestigings e-mail krijgen met de informatie over je reservatie.

Hoeveel op voorhand moet ik reserveren?

Er staat geen tijd op hoe ver op voorhand je moet reserveren. Maar wacht niet te lang met reserveren want de plaatsjes zijn rap gevuld!

Contact
Kan ik jullie vinden op facebook?

Je kan ons vinden op de naam Virtual Spectrales. Volg ons zeker. Zo blijf je op de hoogte van de nieuwste unieke ervaringen die je bij ons kan beleven.

Hoe schrijf ik me in voor de nieuwsbrief?

Onderaan elke pagina kan je je e-mailadres ingeven in het vakje ‘Je e-mailadres’ daarna druk je op ‘inschrijven’ om het te bevestigen. Als je je inschrijft op onze nieuwsbrief krijg je 5 euro korting op de volgende reservatie. Je zal een e-mail ontvangen van ons met een persoonlijke promotiecode in.
